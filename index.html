Omnoctopus

What is it

A framework for additional features for the Java EE Security API specification, which aren't yet available within the specification and RI (Soteria)

Features

Some of the (planned) features :
  • Additional Authentication methods and Identitity Stores based on OAuth2, JWT, ...
  • Events when authenticated, ...
  • Authorization annotations like RequiresUser, ...
  • More finegrained authorization logic with permissions, RBAC, ...
  • ...
